Github Pages + Hugo 搭建个人博客

想快速搭建一个人博客，又不想搞服务器以及域名因此白嫖一下 Github Pages

0、主要步骤

  1. 安装Hugo
  2. 使用Hugo 生成 网站代码，
  3. 把生成的网站代码上传到 github上

一、安装 Hugo 和 Git

本次安装以及调试是在Mac电脑，本地安装使用 home brew 不了解的可以查看下 home brew 官网

Home brew 官网安装

Hugo 官方介绍文档

bres install hugo

➜  ~ brew install hugo #因为已经安装过，就贴一个安装过的截图了

Running `brew update --auto-update`...
Warning: hugo 0.101.0 is already installed and up-to-date.
To reinstall 0.101.0, run:
  brew reinstall hugo

二、Hugo 创建项目

  1. 创建 GoldPillar 项目
# GoldPillar 是网站文件夹名 
 ~ hugo new site GoldPillar

# 创建成功后提示
Congratulations! Your new Hugo site is created in /Users/Max/Desktop/GoldPillar.

Just a few more steps and you're ready to go:

1. Download a theme into the same-named folder.
   Choose a theme from https://themes.gohugo.io/ or
   create your own with the "hugo new theme <THEMENAME>" command.
2. Perhaps you want to add some content. You can add single files
   with "hugo new <SECTIONNAME>/<FILENAME>.<FORMAT>".
3. Start the built-in live server via "hugo server".

Visit https://gohugo.io/ for quickstart guide and full documentation.

项目目录如下

.
├── archetypes
│   └── default.md
├── config.toml
├── content
├── data
├── layouts
├── public
├── static
└── themes

7 directories, 2 files

选择主题

有需要的可以去主题挑选一款自己喜欢的Hugo Themes

cd GoldPillar/themes
git clone https://github.com/dillonzq/LoveIt.git

添加主题到 config.toml

echo "theme = 'LoveIt'" >> config.toml

toml内容如下

baseURL = 'https://Q14.github.io'
languageCode = 'zh-CN'
title = 'Goldpillar joe 的博客'
theme = 'LoveIt'

三、Hugo 创建Blog

hugo new post/first.md

GoldPillar/content/post/ 目录下是你创建的blog

四、生成html代码&部署到Github

hugo
cd public 
# public 目录如下
├── categories
│   └── index.xml
├── index.xml
├── sitemap.xml
└── tags
    └── index.xml
2 directories, 4 files

在public 目录下git init

github 上创建仓库 如下

创建github 同名userName.github.io

public 代码退到远程github 仓库上

git init 
git add .
git commit -m 'crate Q14 github.io site'
git remote add origin git@github.com:Q14/q14.github.io.git
git push -uf origin main

五、成功创建 https://q14.github.io/ 打开看看吧
